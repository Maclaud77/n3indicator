# Changelog — N3 Indicator

## [1.0.0] - 2026-07-14

### Added
- Badge N3 con color configurable en el listado de tickets
- Banner visual dentro del ticket N3
- Página de configuración para grupos y colores
- Soporte para múltiples grupos N3 con colores individuales
- Activar/desactivar grupos sin perder configuración
- Compatible con GLPI 11.0.x
